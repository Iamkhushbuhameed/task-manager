# Task Manager App

A simple and efficient Task Manager built using *Next.js* and *TypeScript*. This app allows users to add, edit, delete, toggle tasks with a clean and responsive interface. register and login as well

## Back-End Implementation
- JWT Authentication
- Protected Routes
- CRUD Tasks
- Backend: Node + Express + TypeScript
- Frontend: Next.js App Router
- API tested(Thunder + UI)   

## Features
- Implement Register And Login
- Add new tasks with title and description
- Edit existing tasks
- Delete tasks
- Toggle tasks
- Mark tasks as completed

## Technologies Used
-  Frontend: Next.js, TypeScript
-  Data Storage: localStorage
